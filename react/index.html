REACT

React는 Facebook에서 개발한 JavaScript 라이브러리로, 사용자 인터페이스를 만드는 데 사용됩니다.

React는 Facebook에서 개발한 JavaScript 라이브러리로, 사용자 인터페이스를 만드는 데 사용됩니다. React는 컴포넌트 기반 아키텍처를 사용하며, 각 컴포넌트는 자체적으로 상태를 관리하고, UI 렌더링 로직을 포함합니다.

React는 Virtual DOM 기술을 사용하여 성능을 최적화합니다. 이는 DOM 조작 작업을 최소화하고 변경된 부분만 업데이트함으로써 불필요한 렌더링 작업을 방지합니다. 또한 React는 다른 라이브러리와의 연동성이 뛰어나며, React Native를 통해 네이티브 앱 개발도 가능합니다.

2. JSX

JSX는 자바스크립트의 확장 문법으로, React.js에서 UI를 작성할 때 사용됩니다. JSX는 HTML과 유사한 구문을 사용하여 UI를 렌더링할 수 있도록 하며, React에서는 JSX 코드를 자바스크립트 함수로 변환하여 사용합니다.

JSX는 일반적으로 HTML과 유사한 태그와 속성을 사용하여 컴포넌트를 정의하며, 이를 통해 개발자는 React 컴포넌트를 작성할 때 보다 직관적이고 가독성이 높은 코드를 작성할 수 있습니다. 또한, JSX는 컴파일 과정에서 자바스크립트로 변환되므로, 브라우저에서 직접 실행될 수 있으며, 성능에도 영향을 미치지 않습니다.

import React from "react";
import ReactDOM from "react-dom/client";

const name = "jung";
const hello = <h1>hello {name}</h1>;

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(hello);

//hello jung

객체와 함수를 같이 사용한 경우

import React from "react";
import ReactDOM from "react-dom/client";

function helloName(){
    return name.nick
}
  
const name = {
    nick : "jung",
}
  
const hello = <h1>Hello, {helloName()}</h1>

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(hello);

//hello jung
